In Brief:

Code Camp Houston is a free day of sessions relating to software development organized by the 2011 Committee. The event is free to attend, but registration is required.

A Code Camp follows these rules:

  • They are meant to be a place for developers to come and learn from their peers. Topics are always based on community interest and never determined by anyone other than the community.
  • Code Camps are always FREE for attendees.
  • The success of the Code Camps is that they are based on community content. All content that is delivered is original. All presentation content must be provided completely (including code) without any restriction.
  • No Fluff â€“ Code Code Camps are about showing the code.
  • All are welcome to attend and speak and do so without expectation of payment.
  • The beauty of the Code Camp is that they always occur on weekends.

Speaker Information:

If you would like to contribute to the community and to Code Camp 2011, then get in contact with us. Make sure you specify on the contact form that you are interested in speaking and be sure to give us a hint as to what your talk will be about. Someone from our side will contact you shortly. Be sure to have further information (such as a photo, job title, work experience) ready to go so that we can add it to our speaker page and to the official schedule.

Sponsor Information:

Fill out the form if you would be interested in sponsoring Code Camp. Fill out the proper fields and we will get in contact with you. Be prepared to have a logo/brand ready for us so that we can list you under our sponsors section on the homepage.

  • Platinum Sponsors are sponsors who have donated $500 of services to Code Camp
  • Gold Sponsors are sponsors who have donated $300 of services to Code Camp
  • Silver Sponsors are sponsors who have donated $100 of services to Code Camp
